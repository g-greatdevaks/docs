# .NET - Queue.Enqueue()

Send messages to a queue.

```csharp
class UserRequest
{
    public string UserId { get; set; }
    public string Message { get; set; }
}

var queue = Nitric.Sdk.Nitric.Queue<UserRequest>("batch").With(QueuePermission.Enqueue);

                QueuePermission.Enqueue,
                QueuePermission.Dequeue

queue.Enqueue(new UserRequest
{
    UserId = "1234",
    Message = "Hello!"
});

Nitric.Sdk.Nitric.Run();
```

## Parameters

<Properties>
  <Property name="tasks" required type="Task<T> | Task<T>[] | T | T[]">
    The task or an array of tasks to send to the queue.

## Examples

### Send a task to a queue

```csharp
using System.Collections.Generic;

class UserRequest
{
    public string UserId { get; set; }
    public string Message { get; set; }
}

var queue = Nitric.Queue<UserRequest>("batch").With(QueuePermission.Enqueue);

queue.Enqueue(
    new Task<UserRequest>
    {
        Id = "1234",
        PayloadType = "user_request",
        Payload = new UserRequest
        {
            UserId = "user_1234",
            Message = "Hello!"
        }
    }
);

Nitric.Sdk.Nitric.Run();
```

### Send multiple tasks to a queue

```csharp
using System.Collections.Generic;

var queue = Nitric.Queue("batch").With(QueuePermission.Enqueue);

queue.Enqueue(
    new Task[]
    {
        new UserRequest
        {
            UserId = "user_1",
            Message = "Batch Task 1"
        }, new UserRequest
        {
            UserId = "user_2",
            Message = "Batch Task 2"
        }
    }
);

Nitric.Sdk.Nitric.Run();
```

### Dealing with failures

In rare cases when sending tasks to a queue some tasks might fail to be sent. The response from `send()` will include an array of any tasks that failed to send. You can process this array to retry or log the error.

```csharp
var failed = await queue.Enqueue(tasks);

failed.ForEach(failedTask => {
    Console.WriteLine(failedTask.Message);
});

Nitric.Sdk.Nitric.Run();
```
