# .NET - KV.Get()

Get a value from a key value store.

```csharp
var kv = Nitric.Sdk.Nitric.KeyValue<Profile>("profiles").With(
    KeyValueStorePermission.Getting
);

var profile = kv.Get('profile-1a2b3c');
```

## Parameters

<Properties>
  <Property name="key" required type="string">
    The key that references the key value pair that should be retrieved.
  </Property>
</Properties>
