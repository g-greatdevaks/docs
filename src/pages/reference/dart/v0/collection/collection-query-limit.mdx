# Dart - collection.query.limit()

Limit the number of results returned by a query.

```dart
import { collection } from '@nitric/sdk'

const profiles = collection('profiles').for('reading')

const profileQuery = profiles.query().limit(10)
```

## Parameters

<Properties>
  <Property name="limit" required type="number">
    The limit to apply to the query.
  </Property>
</Properties>
